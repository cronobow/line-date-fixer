# LINE Chat 日期補齊工具 (LINE Date Fixer)

> 🔗 **線上工具網址**：[https://line-date-fixer.gh.maxlab.tw](https://line-date-fixer.gh.maxlab.tw)

本工具是一個純前端的網頁工具，旨在解決 LINE 匯出對話紀錄文字檔（`.txt`）時，日期只顯示在一天開頭、而後續訊息僅有時間（如「上午 10:30」）的痛點。

本工具會自動解析對話紀錄，並在每條訊息的時間前方**標記上所屬日期**（例如將「上午 10:30 傳送的訊息」自動補齊為 `2025-09-22 10:30`）。**本工具同時支援一般 LINE 聊天群組以及 LINE 社群 (OpenChat)** 的匯出文字格式。

---

## 💡 為什麼需要這個工具？ (AI / NotebookLM 協作神器)

當我們將原始的 LINE 對話紀錄直接匯入至 **NotebookLM**、**ChatGPT**、**Claude** 或 **Gemini** 等 AI 工具時，由於原始檔案缺乏每行訊息的日期資訊，AI 往往無法精確比對對話發生的天數與先後順序，容易造成時間上的混亂。

透過本工具處理後，每行對話都具備明確的 `YYYY-MM-DD HH:MM` 時間戳記，AI 能夠非常精準地為您進行時序分析、事件檢索與摘要整理。

## ✍️ NotebookLM / AI 實用提示詞 (Prompt) 範例

將處理後的對話文字檔上傳至 NotebookLM 後，您可以嘗試使用以下提示詞來快速整理與查找資料：

### ✈️ 範例一：旅遊規劃與生活回憶
> **Prompt:**
> 「請幫我找出 2025 年所有關於旅行計畫的對話，並整理出討論過程的時間軸、景點提議以及最終決定的行程摘要。」

### 🍕 範例二：家庭或朋友群組 / 聚餐規劃與分攤費用
> **Prompt:**
> 「請幫我搜尋 2026 年春節期間，大家在群組裡討論聚餐的所有訊息。請幫我整理出最後決定去的餐廳、聚餐時間，以及當時大家討論如何分攤與結算費用的紀錄。」

### 🛍️ 範例三：專業社群或興趣群組 / 推薦清單與好物整理
> **Prompt:**
> 「請分析這份『露營裝備推薦社群』的對話紀錄，整理出大家在 2025 年期間推薦的『充氣床墊』品牌與款式清單。請幫我列出每個品牌的推薦原因、大致價位，以及社群成員們的使用評價心得。」

## 🛠️ 如何使用

1. **開啟網頁**：請直接至 [LINE Date Fixer 線上工具](https://line-date-fixer.gh.maxlab.tw) 開啟使用。
2. **匯入檔案**：將從 LINE 匯出的對話紀錄 `.txt` 檔案拖放至頁面中的拖曳區，或點擊選取檔案。
3. **下載結果**：系統會立即在瀏覽器中完成解析，顯示處理摘要與前 80 行預覽，並提供下載按鈕。點擊下載即可獲得檔名後綴為 `_processed.txt` 的新檔案。

---

## 🔒 隱私與安全聲明

本工具為**純前端實作**，所有的檔案讀取、字串解析及新檔生成皆在您的**本機瀏覽器**內完成。**絕對不會**將您的任何對話紀錄上傳至任何後端伺服器或外部網路，確保您的個人對話私密安全。
